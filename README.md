<h1 align="center">Telegram-Minigame-Demo</h1>

A basic Tap-to-Earn (T2E) Telegram minigame, where users can complete quests or tap on funny 
Elon Musk coin to receive `$Musktap` points. Using points, they can buy boosters or purchase 
surprise boxes, each containing random amount of `$MUSK` token. MuskTap is an airdrop platform, 
similar to HamsterKombat or NotCoin.

- Open Telegram bot, you will receive random $MUSK tokens, a giveaway that might even worth **10$** üí∞
- Tap to earn $MuskTap points. They are used to improve profile or purchase items. ü•Ø
- Buy boosters using your points. You can get more points by each tap, tap more, etc. ‚ö°
- Purchase MUSK boxes. Boxes will be opened later, each including random amount of $MUSK (or other exciting perks!) üéÅ
- Share your referral code with friends to earn more $MuskTap points üë•
